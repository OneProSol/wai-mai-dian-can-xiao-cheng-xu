### 一、相关技术
- 前端：`jsp`、`微信小程序`。
- 后端：`Springboot`等。
- 数据库：`MySQL`

### 二、开发软件（列出的软件其一均可运行）
- `IDEA`
- `Eclipse`
- `Visual Studio Code(VScode)`
- `Navicat`
### 三、部署所需环境

- 后端：`jdk1.8`、`阿里云oss`。
- 前端：`微信小程序开发工具`  。
- 数据库：`mysql5.7以上`

### 四、功能描述及功能图展示
系统分为：用户、管理员。

用户功能：1. 点菜、2. 商家、3. 购物车、4. 订单、5. 个人中心

管理员功能：1. 登录、2. 新增商品、3. 所有商品、4. 菜单管理、5. 菜单分类、6. 订单总览、7. 收入统计、8. 评价总览、9. 店铺信息
